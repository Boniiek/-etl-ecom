ETL приложение для расчета топ-3 магазинов по городам на основе заказов пользователей, зарегистрированных в 2025 году.

* Создайте файл .env в корневой директории проекта:

S3_ENDPOINT=http://minio:9000 \
S3_ACCESS_KEY=etl \
S3_SECRET_KEY=etladmin123\
S3_BUCKET_NAME=etl-data\

* Запуск приложения: \
docker-compose up --build -d


* После запуска приложения: 

Загрузить данные для обработки 
в MinIO Console: http://localhost:9001

Логин: etl \
Пароль: etladmin123

Загрузите ваши Parquet файлы в соответствующие папки:

store/ - данные магазинов

user/ - данные пользователей

order/ - данные заказов

* Запуск ETL процесса:\

docker-compose run --rm etl-app

* Результат появится в папке result/ в MinIO Console:

Важно:
1) Файлы должны быть в формате Parquet

2) Папки должны называться строго: store/, user/, order/

3) Результат сохраняется в папку result/

4) Каждый запуск перезаписывает предыдущий результат

5) Для корректной работы необходимы все три источника данных
